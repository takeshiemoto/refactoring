# vue-tour-of-heroes
Tour of heroes for Vue.js
 
## Routing
|URL|Component|Description|
|---|---|---|
|/dashboard|Dashboard.vue|GET|
|/heroes  |Heroes.vue|GET / CREATE / DELETE|
|/hero/:id|HeroDetail.vue|UPDATE|

## Entity
### Hero
|No|Name|Type|
|---|---|---|
|1|id|string|
|2|name|string|
|3|created|date|
|4|updated|date|
